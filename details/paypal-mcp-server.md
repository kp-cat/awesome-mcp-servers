## Overview

PayPal's official MCP server allows merchants to integrate PayPal's full suite of APIs through the Model Context Protocol, enabling AI agents to manage the complete payment lifecycle. Access tools for orders, invoices, disputes, shipment tracking, transaction search, and subscriptions.

## Features

### Payment Operations
- **Order Management**: Create, update, capture, and refund orders
- **Invoice Operations**: Generate, send, and track invoices
- **Transaction Search**: Query and analyze payment history
- **Subscription Management**: Create and manage recurring billing

### Customer Service
- **Dispute Resolution**: View and respond to payment disputes
- **Shipment Tracking**: Update and monitor order fulfillment
- **Refund Processing**: Handle returns and refunds
- **Payment Status**: Check transaction and order status

### Business Intelligence
- Transaction reporting and analytics
- Revenue tracking across time periods
- Customer payment insights
- Subscription metrics and churn analysis

## Integration

Works with AI assistants supporting MCP. Requires PayPal business account and API credentials (Client ID and Secret). Supports both sandbox and production environments.

## Pricing

PayPal charges transaction fees based on payment type and volume. The MCP server itself is provided free to PayPal merchants. Standard PayPal fees apply to all processed transactions.
