## Overview

Smithery is the closest thing to a homepage for MCP servers, providing the leading directory and hosting platform for the Model Context Protocol ecosystem. With over 2,000 registered servers, it serves as the primary discovery and distribution channel for MCP implementations.

## Platform Features

- **MCP Server Directory**: Comprehensive registry of 2,000+ MCP servers
- **Built-in Infrastructure**: Ready-to-use hosting and distribution
- **Developer Tools**: FastMCP integration for rapid Python development
- **Smithery Playground**: Interactive testing and development environment
- **HTTP Access**: Deploy servers with HTTP endpoints
- **Discovery Services**: Search by keyword or natural language
- **Agent Toolbox**: Runtime server search for AI agents

## Developer Experience

- **FastMCP Integration**: Build Python MCP servers with custom Docker containers
- **Rapid Iteration**: Test and deploy quickly through Smithery Playground
- **One-Click Deployment**: Simple deployment to HTTP endpoints
- **Documentation**: Comprehensive guides and quickstarts
