## Overview

Zoom's MCP integration enables app developers to leverage the Model Context Protocol to connect Zoom apps to remote MCP servers, build new MCP-powered applications in AI Studio, and create custom agents with third-party integrations.

## Features

- **Remote MCP Server Connection**: Connect Zoom apps to external MCP servers
- **AI Studio Integration**: Build MCP-powered apps using Zoom's AI Studio
- **Custom Agent Development**: Create agents that connect to third-party tools (Jira, Box, etc.)
- **Zoom App Marketplace**: Publish and distribute MCP-enabled Zoom apps
- **Enhanced Connectors**: Integrate with CRM systems like Salesforce for real-time meeting insights

## Capabilities

- Query customer data during meetings (e.g., Salesforce integration)
- Provide real-time insights from external systems
- Automate meeting workflows with third-party tools
- Build custom AI agents for specific use cases

## Use Cases

- Sales call assistance with CRM data access
- Meeting automation with project management tools
- Customer support with ticket system integration
- Real-time data insights during video conferences

## Admin Features

Admins can build and deploy custom agents that connect directly to third-party tools using the Model Context Protocol.

## Pricing

Available as part of Zoom's platform. Specific features may require Zoom AI or enterprise subscriptions.
