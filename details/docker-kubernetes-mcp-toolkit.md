## Overview

The Docker MCP Toolkit with Kubernetes MCP Server provides a powerful yet simple way to manage Kubernetes clusters directly from Docker Desktop environments. In just 5 minutes, you can access 40 specialized tools covering everything from basic resource management to advanced operations like Helm chart deployments and port forwarding.

## Features

### 40+ Specialized Tools

#### Container Management
- List and inspect containers
- Start, stop, and restart containers
- View container logs
- Execute commands in containers
- Manage container lifecycle

#### Image Operations
- Pull and push images
- Build images from Dockerfiles
- Tag and remove images
- Inspect image details
- Manage image layers

#### Kubernetes Integration
- Resource management (pods, deployments, services)
- Namespace operations
- ConfigMap and Secret handling
- Service exposure and ingress
- Scaling and updates

#### Advanced Operations
- **Helm Charts**: Deploy and manage Helm releases
- **Port Forwarding**: Forward ports to local machine
- **Resource Monitoring**: Check cluster health and metrics
- **Log Aggregation**: Collect logs from multiple pods
