## Overview

A community-built MCP server addressing the limitation of read-only access in existing Shopify MCP servers by providing full write operations for order management, refunds, product updates, and payment processing.

## Features

- **Write Operations**: Full create, update, delete capabilities for Shopify resources
- **Order Management**: Process orders, updates, and fulfillments
- **Refund Processing**: Handle refunds and payment reversals
- **Product Management**: Create and update product catalogs
- **Multi-Store Support**: Manage multiple Shopify stores
- **Stripe Integration**: Payment processing with Stripe

## Capabilities

- Create and modify orders
- Process refunds and cancellations
- Update product information and inventory
- Manage customer data
- Handle payment operations via Stripe
- Support for multiple store configurations

## Use Cases

- Automated order processing workflows
- AI-powered refund management
- Dynamic product catalog updates
- Multi-store e-commerce management
- Payment automation and reconciliation

## Difference from Official Server

While Shopify's official MCP servers focus on development resources and read operations, this community server provides full write access for commerce operations.

## Pricing

Free and open-source. Shopify and Stripe accounts required.
