## Overview

Raycast MCP Server provides comprehensive integration between AI assistants and Raycast, the powerful productivity launcher for macOS. It offers 9 specialized tools for automating and controlling Raycast features through natural language.

## Available Tools

### 1. Raycast Auth
- Manage Raycast authentication and sessions
- API token configuration
- User profile access
- Permission management

### 2. Raycast Extensions
- Browse and discover Raycast extensions
- Install and uninstall extensions
- Configure extension settings
- Access extension store
- Manage extension updates

### 3. Raycast Workflows
- Create custom automation workflows
- Manage workflow triggers
- Configure workflow actions
- Chain multiple operations
- Conditional workflow logic

### 4. Raycast Search
- Search applications and launch them
- File and document search
- Web search integration
- Clipboard history search
- Quick calculations

### 5. Raycast Clipboard
- Access clipboard history
- Manage clipboard items
- Search clipboard content
- Pin frequently used items
- Clipboard synchronization

### 6. Raycast Shortcuts
- Trigger Raycast shortcuts programmatically
- Configure hotkey bindings
- Manage global shortcuts
- Custom shortcut creation

## Platform

Designed specifically for macOS, leveraging Raycast's native features and performance optimizations.

## Technical Implementation

Built to integrate directly with Raycast's APIs and extension system, providing seamless access to all Raycast features through the Model Context Protocol.

## Integration

Compatible with Claude Desktop, Cursor, and other MCP clients running on macOS. Requires Raycast installed and configured on the system.
