## Overview

Gong has introduced Model Context Protocol (MCP) support to unify enterprise AI agents from HubSpot, Microsoft, Salesforce, and other major platforms, enabling seamless revenue intelligence integration across business tools.

## Features

- **Multi-Platform Integration**: Unifies AI agents from HubSpot, Microsoft, Salesforce, and others
- **Revenue Intelligence**: Connects Gong's conversation analytics with external systems
- **Standardized Interface**: Uses MCP for consistent cross-platform communication
- **Enterprise AI Orchestration**: Coordinates multiple AI agents across business platforms
- **Data Synchronization**: Keeps revenue data in sync across tools

## Supported Integrations

- HubSpot CRM
- Microsoft Dynamics 365
- Salesforce
- Additional enterprise platforms through MCP

## Use Cases

- Unified revenue intelligence across CRM systems
- Cross-platform sales analytics
- Automated deal insights and recommendations
- Multi-system workflow automation
- Enterprise-wide AI agent coordination
