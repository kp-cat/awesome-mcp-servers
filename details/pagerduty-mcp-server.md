## Overview

The PagerDuty MCP Server is the official Model Context Protocol implementation from PagerDuty, enabling AI assistants and language models to interact directly with your PagerDuty account for incident management, on-call scheduling, and service operations.

## Deployment Options

### PagerDuty-Hosted Service
Managed service hosted at mcp.pagerduty.com/mcp

### Self-Hosted
Run your own MCP server using PagerDuty's open-source implementation

## Key Capabilities

- **Incident Management**: Real-time access to incidents, outlier incidents, past incidents, related incidents
- **On-Call Schedules**: Query and manage on-call rotations
- **Service Operations**: Interact with services and configurations
- **Teams & Escalation**: Manage teams and escalation policies
- **Event Orchestration**: Access service and global event orchestration configurations
- **Incident Workflows**: Automated incident response workflows
- **Status Pages**: Monitor and update status pages
