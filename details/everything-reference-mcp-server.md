## Overview

The Everything Reference MCP Server is Anthropic's comprehensive test and reference implementation that demonstrates all aspects of the Model Context Protocol. It serves as both a testing tool and educational resource for MCP developers.

## Purpose

- **Testing**: Validate MCP client implementations
- **Reference**: Demonstrate protocol features and best practices
- **Education**: Learn MCP capabilities through examples
- **Development**: Test bed for new MCP features

## Features

- **Prompts**: Example prompt templates and patterns
- **Resources**: Various resource types and access patterns
- **Tools**: Comprehensive tool implementations
- **All Protocol Features**: Complete MCP specification coverage
- **Edge Cases**: Handling of unusual scenarios
- **Error Examples**: Proper error handling demonstrations
