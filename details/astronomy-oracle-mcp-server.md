## Overview

Astronomy Oracle is a specialized MCP server designed for astronomers and astronomy enthusiasts, providing comprehensive astronomical data and observing session planning capabilities through AI assistants.

## Features

- **Deep-Sky Object Catalog**: Access to 13,000+ objects from OpenNGC
- **Visibility Calculations**: Deterministic visibility computations for celestial objects
- **Rise/Transit/Set Times**: Accurate calculations for object observation timing
- **Altitude/Azimuth Data**: Precise positional data for sky objects
- **Observing Session Planner**: AI-powered session planning for optimal observations
- **Accurate Astronomical Data**: Verified catalog data for scientific use

## Data Sources

- OpenNGC catalog (13,000+ deep-sky objects)
- Astronomical calculations based on standard algorithms
- Coordinate transformations and time systems

## Use Cases

- Planning astronomical observing sessions
- Finding optimal viewing times for celestial objects
- Educational astronomy projects
- Amateur astronomy automation
- Astrophotography planning
- Telescope automation and control
- Astronomy research and data analysis

## Calculations Provided

- Object visibility from given location and time
- Rise, transit, and set times
- Altitude and azimuth coordinates
- Optimal viewing periods
- Object separation and proximity

- Amateur astronomers
- Professional observatories
- Astrophotographers
- Astronomy educators and students
- Planetarium software developers
