## Overview

Plain is one of three customer support platforms with an official, first-party MCP server (alongside Intercom and Pylon). The Plain MCP Server exposes 30 tools across five domains, enabling AI assistants to read and write customer support data with robust OAuth 2.0 with PKCE authentication.

## Features

### 30 Tools Across 5 Domains
- **Ticket Management**: Create, update, assign, and close support tickets
- **Customer Operations**: Manage customer records and interactions
- **Workflow Automation**: Automate support processes and routing
- **Team Management**: Assign work and manage team capacity
- **Reporting & Analytics**: Track metrics and team performance

### Security & Authentication
- **OAuth 2.0 with PKCE**: Industry-standard secure authentication
- **Scoped Permissions**: Granular access control
- **Audit Logging**: Track all AI-initiated actions
- **Data Privacy**: Compliant with GDPR and data protection regulations

### Integration Capabilities
- Read and write access to support data
- Real-time ticket updates
- Customer context enrichment
- Team collaboration features

## Platform Comparison

As of March 2026, only three customer support platforms have official MCP servers:
- **Plain**: 30 tools, full OAuth support, read/write access
- **Intercom**: 6 tools, US-hosted workspaces only, limited functionality
- **Pylon**: 6 tools, read/write access to issues and accounts

Major platforms like Zendesk, Freshdesk, Help Scout, Front, and HubSpot Service Hub do not have official MCP servers.

## Integration

Works with AI assistants supporting MCP including Claude, Cursor, and other MCP-compatible tools. Requires Plain account and OAuth authentication setup.

## Pricing

Plain offers usage-based pricing with a free trial. MCP server access included with all paid plans.
