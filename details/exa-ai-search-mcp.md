## Overview

Exa AI provides neural semantic search capabilities specifically designed for AI applications. Unlike traditional keyword-based search, Exa uses semantic understanding to find relevant information across diverse sources including academic papers, company data, and developer platforms.

## Features

- **Neural Semantic Search**: AI-powered understanding of query intent and context
- **Multi-Source Coverage**: Search across academic papers, LinkedIn, Wikipedia, GitHub, company data
- **Research-Optimized**: Specialized for research-style queries and deep investigation
- **AI-Native Design**: Built from the ground up for LLM consumption
- **Embedding-Based**: Uses vector embeddings for superior relevance
- **Developer-Friendly**: Comprehensive API for integration with AI workflows

## Key Advantages

- Better for research-style queries compared to general-purpose search
- Understands semantic relationships and context
- Finds relevant information that keyword search might miss
- Optimized for technical and academic content discovery

## Use Cases

- Academic research and literature review
- Competitive intelligence gathering
- Technical documentation discovery
- Company and market research
- Developer tool and library discovery
