## Overview

The Shopify Customer Accounts MCP server lets AI assistants help customers with order tracking, returns, and account information. This server complements the Storefront MCP and Dev MCP servers to provide a complete Shopify integration for AI-powered customer experiences.

## Features

### Order Management
- **Order Tracking**: Check order status and shipping information
- **Order History**: View past purchases and order details
- **Return Requests**: Initiate and track product returns
- **Refund Status**: Check refund processing and status

### Account Operations
- **Profile Management**: View and update customer information
- **Address Book**: Manage shipping and billing addresses
- **Payment Methods**: View saved payment information
- **Account Settings**: Update preferences and notifications

### Customer Service
- **Order Issues**: Report problems with orders
- **Support Tickets**: Create and track customer support requests
- **FAQ Assistance**: Get answers to common questions
- **Product Questions**: Ask about products and policies

## Shopify MCP Ecosystem

Shopify provides three complementary MCP servers:
- **Storefront MCP**: For product browsing and purchasing
- **Customer Accounts MCP**: For post-purchase support and account management
- **Dev MCP**: For accessing Shopify documentation and API reference

## Integration

Works with AI assistants supporting MCP including Claude, Cursor, and other tools. Requires Shopify store and appropriate API credentials. Respects customer privacy and permission settings.

## Pricing

Included with all Shopify plans. No additional cost for using the MCP server with your Shopify store.
