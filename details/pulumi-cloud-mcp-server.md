## Overview

The Pulumi MCP Server allows developers to provision cloud infrastructure by issuing natural language instructions to AI assistants, which then execute Pulumi CLI commands. It supports multi-cloud deployments and modern infrastructure-as-code practices.

## Features

- **Multi-Cloud Support**: Deploy to AWS, Azure, Google Cloud, Kubernetes, and 100+ providers
- **Natural Language Provisioning**: Describe infrastructure in plain English
- **Pulumi CLI Integration**: Direct execution of Pulumi commands
- **State Management**: Automatic handling of infrastructure state
- **Preview Changes**: Review infrastructure changes before deployment
- **Programming Language Support**: Use TypeScript, Python, Go, C#, Java

## Capabilities

- Provision cloud resources through conversational commands
- Deploy AKS, EKS, GKE clusters via natural language
- Manage infrastructure stacks
- Update and destroy resources
- Query stack outputs and status
- Import existing infrastructure

## Example Use Cases

**User**: "Provision an Azure Kubernetes Service cluster with 3 nodes"

The MCP server translates this to appropriate Pulumi commands and executes the deployment.
