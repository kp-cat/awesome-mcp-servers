## Overview

The official Redis MCP Server offers a natural language interface designed for agentic applications to manage and search data in Redis efficiently. Additionally, the Redis Cloud API MCP Server allows you to manage your Redis Cloud resources using natural language, providing comprehensive Redis integration for AI agents.

## Features

### Data Operations
- **Key-Value Operations**: Get, set, delete Redis keys
- **Data Structures**: Lists, sets, sorted sets, hashes, streams
- **Search**: Full-text search with RediSearch
- **JSON**: Native JSON document storage and queries
- **Time Series**: Time-series data management

### Redis Cloud Management
- **Resource Management**: Create and manage Redis databases
- **Subscription Control**: Manage Redis Cloud subscriptions
- **Monitoring**: Track performance metrics
- **Configuration**: Adjust database settings

### Advanced Capabilities
- **Pub/Sub**: Real-time messaging
- **Streams**: Event streaming and processing
- **Vector Search**: Similarity search for AI/ML
- **Graph**: Graph database operations with RedisGraph
