## Overview

IO Aerospace MCP Server is a specialized .NET-based implementation providing comprehensive aerospace and astrodynamics capabilities through AI assistants, enabling space mission planning and orbital mechanics calculations.

## Features

- **Ephemeris Calculations**: Precise celestial body position and motion data
- **Orbital Conversions**: Transform between various orbital element systems
- **Deep Space Station Tools**: DSS operations and communication planning
- **Time Conversions**: Convert between different time scales (UTC, TDB, TT, etc.)
- **Unit Conversions**: Aerospace-specific unit transformations
- **Mathematical Utilities**: Specialized math functions for orbital mechanics

## Capabilities

### Orbital Mechanics
- Keplerian orbital elements
- State vector conversions
- Orbital maneuver planning
- Transfer orbit calculations
- Rendezvous and proximity operations

### Mission Planning
- Launch window calculations
- Ground station visibility
- Communication link analysis
- Trajectory optimization

### Time Systems
- UTC to TDB conversions
- Julian Date calculations
- Epoch transformations
- Time zone handling for ground operations
