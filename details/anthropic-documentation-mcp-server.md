## Overview

The Anthropic Documentation MCP Server provides AI assistants with access to Anthropic's official documentation, including Claude AI API documentation, Model Context Protocol specifications, and development best practices. This enables developers to get up-to-date information about building with Claude directly from their AI coding assistants.

## Features

### Documentation Access
- **Claude API Documentation**: Complete API reference for Claude
- **MCP Specifications**: Model Context Protocol technical docs
- **Getting Started Guides**: Tutorials and quickstart guides
- **Best Practices**: Recommendations for building with Claude
