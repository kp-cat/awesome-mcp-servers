## Overview

The Atlassian Rovo MCP Server is the official cloud-based integration from Atlassian that serves as a bridge between Atlassian Cloud products and external AI tools. It enables real-time interaction with Jira, Compass, and Confluence through natural language.

## Supported Products

- **Jira**: Issue tracking and project management
- **Confluence**: Team collaboration and documentation
- **Compass**: Software component catalog and health tracking

## Features

- **Content Search**: Search across Jira, Compass, and Confluence without switching tools
- **Natural Language Commands**: Create and update issues/pages via conversational interface
- **Workflow Automation**: Automate repetitive tasks like generating tickets from meeting notes
- **Real-Time Sync**: Always current with latest Atlassian Cloud data
- **Secure Connection**: Enterprise-grade security and authentication
- **Cloud and Server Support**: Works with both Cloud and Server/Data Center deployments

## Capabilities

- Summarize and search Jira issues
- Create new issues and tickets
- Update existing Confluence pages
- Query Compass component data
- Automated ticket triage and routing
- Generate documentation from specs
- Cross-product data synthesis

## Use Cases

- AI-assisted project management
- Automated ticket creation from requirements
- Documentation generation and updates
- Issue triage and categorization
- Sprint planning assistance
- Knowledge base management
- Cross-team collaboration automation

## Integration

Works with LLMs, IDEs, and agent platforms including Claude, Cursor, and other MCP-compatible clients.
