## Overview

With MCP support in Agentforce 3, Salesforce provides a native MCP client that dramatically simplifies how AI agents connect to the wider enterprise technology stack. This plug-and-play capability allows you to expose business logic, data, and workflows to AI agents so they can act, not just observe—turning any API, integration, or system into an action the agent can invoke without custom code.

## Features

### Native MCP Client
- **Built-in Integration**: MCP client native to Agentforce 3
- **No Custom Code**: Connect to MCP servers without development
- **Plug-and-Play**: Simple configuration to enable capabilities
- **Automatic Discovery**: Agents discover available MCP tools

### Business Logic Exposure
- **Salesforce Data**: CRM data access through MCP
- **Custom Objects**: Expose custom Salesforce objects
- **Apex Methods**: Make Apex code available to agents
- **External Systems**: Connect to non-Salesforce systems

### Workflow Integration
- **Flow Actions**: Trigger Salesforce Flows from agents
- **Approval Processes**: Manage approvals conversationally
- **Automation**: Execute business process automation
- **Event Handling**: Respond to platform events
