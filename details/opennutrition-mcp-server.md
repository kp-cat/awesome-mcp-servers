## Overview

OpenNutrition MCP Server connects AI assistants to a comprehensive database of over 300,000 foods with detailed nutrition information. It enables intelligent meal planning, dietary analysis, and nutrition tracking through natural language.

## Features

### Food Database
- 300,000+ food items
- Detailed nutrition facts
- Macro and micronutrients
- Serving size information
- Brand and generic foods
- Restaurant menu items

### Barcode Lookup
- UPC/EAN barcode scanning
- Product identification
- Instant nutrition facts
- Brand information
- Multi-country support

### Nutrition Information
- Calories and energy
- Macronutrients (protein, carbs, fat)
- Vitamins and minerals
- Fiber and sugar content
- Sodium and cholesterol
- Allergen information

### Search Capabilities
- Natural language food search
- Category filtering
- Brand-specific queries
- Ingredient lookup
- Similar food suggestions

## Use Cases

- Meal planning and recipe creation
- Calorie and macro tracking
- Dietary restriction management
- Nutrition education
- Food logging automation
- Grocery shopping assistance
- Restaurant menu analysis
- Sports nutrition planning
- Weight management support

## AI-Powered Features

### Meal Analysis
- Calculate total nutrition for meals
- Compare food options
- Suggest healthier alternatives
- Balance macronutrient ratios

### Dietary Recommendations
- Goal-based meal suggestions
- Restriction-aware planning (keto, vegan, etc.)
- Allergen avoidance
- Budget-conscious options

### Recipe Analysis
- Parse recipe ingredients
- Calculate nutrition per serving
- Adjust serving sizes
- Ingredient substitutions

## Technical Implementation

Integrates with OpenNutrition database API providing standardized MCP interface. Includes caching for performance and offline capability.

## Data Sources

- USDA FoodData Central
- Commercial food databases
- Restaurant nutritional data
- User-contributed foods
- International food databases

## Integration

Compatible with Claude Desktop, Cursor, VS Code, and other MCP clients. Free tier available with optional premium features for enhanced data access.
