## Overview

The MongoDB MCP Server is the official Model Context Protocol implementation from MongoDB, enabling AI assistants and agents to interact with MongoDB databases and Atlas clusters through natural language commands.

## Features

- **Aggregation Support**: Run complex aggregation pipelines via natural language
- **Schema Exploration**: Describe and inspect collection schemas
- **Index Management**: View and manage database indexes
- **Atlas Integration**: Native support for MongoDB Atlas cloud databases
- **Natural Language Queries**: Convert conversational requests into MongoDB operations
- **Full CRUD Operations**: Create, read, update, and delete documents

## Tools Included

- `aggregate`: Execute aggregation pipelines on collections
- `collection-indexes`: Describe and list collection indexes
- `collection-schema`: Inspect collection schemas and data structures
- Database connection management and authentication
- Query building and optimization assistance

## Use Cases

- AI-assisted database administration
- Natural language database queries
- Automated data analysis and reporting
- Schema design and optimization
- Development workflow automation
