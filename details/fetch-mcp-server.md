## Overview

Fetch is an official reference MCP server maintained by Anthropic that specializes in web content fetching and conversion for AI consumption. It transforms web pages into clean, structured content optimized for large language models.

## Features

- **Web Content Fetching**: Retrieve content from any URL
- **Format Conversion**: Transform HTML to LLM-friendly formats
- **Content Cleaning**: Remove ads, navigation, and irrelevant elements
- **Structured Output**: Organized content for efficient processing
- **Markdown Conversion**: Clean markdown output from web pages
- **Metadata Extraction**: Capture important page metadata

## Capabilities

- Fetch and parse web pages
- Extract main content from complex layouts
- Remove boilerplate and navigation elements
- Convert to markdown or plain text
- Handle JavaScript-rendered content
- Respect robots.txt and rate limiting

## Use Cases
