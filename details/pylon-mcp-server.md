## Overview

Pylon is one of three customer support platforms with an official, first-party MCP server (alongside Plain and Intercom). The Pylon MCP Server provides 6 tools with both read and write access, enabling AI assistants to query and modify support data.

## Features

### 6 Tools for Support Operations
- **query_issues**: Search and filter support issues
- **query_accounts**: Find customer accounts
- **query_users**: Look up user information
- **query_contacts**: Search contact records
- **update_issue**: Modify issue status, priority, and assignments
- **update_account**: Update account information and metadata

### Read & Write Access
- Query support issues with advanced filtering
- Update issue status and assignments
- Modify account information
- Track user and contact data

### Support Workflows
- Issue triage and routing
- Status updates and tracking
- Account management
- User profile access

## Platform Comparison

As of March 2026, only three customer support platforms have official MCP servers:
- **Plain**: 30 tools across 5 domains, most comprehensive
- **Intercom**: 6 tools, US workspaces only, limited write access
- **Pylon**: 6 tools, read and write access, global availability

Major platforms like Zendesk, Freshdesk, and Help Scout lack official MCP servers.
