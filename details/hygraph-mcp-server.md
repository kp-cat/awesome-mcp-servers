## Overview

The Hygraph MCP Server connects your GraphQL-native headless CMS directly to AI assistants using the Model Context Protocol. Manage, query, and publish content through natural language commands in tools like Claude, Cursor, or VS Code.

## Features

### Content Management
- **Schema Discovery**: Explore and understand your content model structure
- **CRUD Operations**: Create, read, update, and delete content entries
- **Bulk Updates**: Modify multiple content entries simultaneously
- **Publishing Workflows**: Manage content stages and publication status

### Developer Tools
- **TypeScript Type Generation**: Automatically generate TypeScript types from your schema
- **GraphQL Query Building**: Construct queries using natural language
- **Localization Support**: Manage multilingual content
- **Asset Management**: Upload and manage media files

### Automation
- Workflow automation for content operations
- Scheduled publishing and unpublishing
- Content migration between environments
- Batch content transformations
