## Overview

Snowflake's managed MCP server (in public preview as of 2026) provides AI agents with an open-standards-based interface to securely access Snowflake data without deploying separate infrastructure.

## Features

- **Managed Service**: No separate infrastructure deployment required
- **Cortex Integration**: Serves Cortex Analyst, Cortex Search, and Cortex Agents as tools
- **Custom Tools**: Support for custom tool development
- **SQL Execution**: Direct SQL query execution through AI agents
- **OAuth Authentication**: Built-in OAuth service for secure authentication
- **Role-Based Access Control**: RBAC for tool discovery and invocation management

## Supported Tools

- **Cortex Analyst**: Natural language data analysis
- **Cortex Search**: Semantic search across data
- **Cortex Agents**: AI-powered data agents
- **Custom SQL Tools**: User-defined SQL operations

## Integration
