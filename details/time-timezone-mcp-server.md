## Overview

The Time & Timezone MCP Server is an official reference implementation from Anthropic that provides comprehensive time and timezone handling capabilities for AI assistants. It enables temporal reasoning and timezone conversions through natural language.

## Features

- **Current Time**: Get current time in any timezone
- **Timezone Conversion**: Convert times between timezones
- **Date Calculations**: Perform date arithmetic and comparisons
- **Format Support**: Multiple time format parsing and output
- **DST Handling**: Automatic daylight saving time adjustments
- **Timezone Database**: Comprehensive IANA timezone database

## Capabilities

- Query current time globally
- Convert between any timezones
- Calculate time differences
- Parse natural language time expressions
- Handle business hours across timezones
- Schedule coordination assistance
- Meeting time suggestions
