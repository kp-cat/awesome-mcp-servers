## Overview

MCP OmniSearch is a comprehensive Model Context Protocol server that provides unified access to multiple search providers and content processing services through a single interface. It eliminates the need to manage separate integrations for different search engines and AI tools.

## Features

- **Multiple Search Engines**: Integration with Tavily, Brave Search, Kagi, and more
- **AI-Powered Tools**: Access to Perplexity and FastGPT for intelligent responses
- **Content Processing**: Built-in support for Jina AI and Kagi content processing
- **Unified Interface**: Single MCP server for all search and processing needs
- **Exa AI Integration**: Neural semantic search optimized for AI applications
- **Firecrawl Support**: Advanced web scraping and content extraction

## Use Cases

- Research tasks requiring multiple search providers
- AI-assisted content discovery and analysis
- Semantic search for academic papers and technical documentation
- Enterprise search across diverse data sources

## Pricing

Free and open-source. Individual API keys required for each integrated service.
