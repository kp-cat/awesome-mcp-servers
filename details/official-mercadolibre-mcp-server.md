## Overview

MercadoLibre's official MCP server provides tools for developers to interact with MercadoLibre's APIs and documentation. The Model Context Protocol (MCP) standardizes connections between AI models and data sources. The official server focuses on API documentation access with tools like search_documentation and get_documentation_page.

## Features

### Documentation Tools
- **search_documentation**: Find relevant API documentation quickly
- **get_documentation_page**: Retrieve specific documentation pages
- **API Reference**: Access complete API specifications
- **Code Examples**: Get implementation samples

## Official vs Community Servers

The official MCP server focuses on documentation and API learning, while community servers (like dan1d/mercadolibre-mcp or lumile/mercadolibre-mcp) provide direct marketplace operations like product search and order management.

## Integration

Works with AI assistants supporting MCP. Provides read-only access to public documentation and API schemas. No API credentials required for documentation access.
